# Python programming concepts

- Python is a `Interpreted language`

    > **Compiled Language** : In a compiled language, the code is translated into machine code before it is run.

    > **Interpreted language** : In an interpreted language, the code is executed line by line by an interpreter, usually written in another language. This makes it easier to write and test code, but it can be slower than compiled languages.

- Code Errors
    - Syntax error
    - Runtime error
    - Logical error
